# Problem 3 --- Remove Duplicates

### Problem Summary

This program removes duplicate numbers from a list of integers and
prints only unique values.

### Algorithm Explanation

1.  Store all numbers in a vector.\
2.  Sort the vector.\
3.  Use the `unique()` function to remove duplicate values.\
4.  Print the remaining elements.

### Time Complexity

O(N log N) because sorting the vector takes this time.

### Space Complexity

O(N) because we store the numbers in a vector.

### Reflection

This problem helped me understand how sorting can help remove duplicates
easily. I also learned how to use STL functions like `sort()` and
`unique()`.
